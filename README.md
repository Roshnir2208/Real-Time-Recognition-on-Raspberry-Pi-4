# Real-Time-Recognition-on-Raspberry-Pi-4
This project implements a real-time computer vision system on a Raspberry Pi 4 designed to detect and identify multiple people and objects within a dynamically changing visual environment. The system combines YOLOv4-tiny for object detection with a HOG-based frontal face detector from the face_recognition framework, after which, 128-dimensional face encodings are used to recognise the four registered identities. Real-time inference produces labelled bounding boxes and confidence scores while tracking individuals as they enter, leave, and re-enter the camera’s field of view. Several optimisations were incorporated to improve performance on embedded hardware, including reduced-resolution processing, periodic detection sampling, persistent detection logic, and scenario-based brightness adjustments tailored to lighting changes.
The system was evaluated across three operating conditions: bright indoor lighting, strong backlighting, and outdoor evening light. For each scenario, a 20-second sequence was recorded, and logs were generated containing temporal presence, detection intervals, and confidence values for all recognised people and objects. Post-processing scripts produced performance plots and quantitative comparisons across scenarios that included logging detection timestamps, confidence scores, and bounding boxes to evaluate the performance. Experimental results show reliable identification of multiple individuals and objects with confidence levels generally above 55–75% for faces and above 65% for objects, with a bottle showing a confidence level of 97%, though performance varies with low-light and backlit conditions. The project demonstrates that careful selection of models, attention to sensor characteristics, and structured logging enable a portable and cost-effective embedded computer vision solution for varied real-world environments.
